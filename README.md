# Contact Book App 📞🐍
A terminal-based Contact Book application built using Python. It saves your contacts (Name and Phone Number) into a text file (`Contacts.txt`) using File Handling and safely reads them using Exception Handling.

## Features
- **Add Contact:** Appends names and phone numbers cleanly into a text file.
- **View Contact:** Displays all saved contacts by reading the text file line-by-line.
- **Error Handling:** Uses `try-except` block to prevent crashes if the file doesn't exist yet.

## How to Run
Run `contacts.py` on Pydroid 3 or any Python compiler.
